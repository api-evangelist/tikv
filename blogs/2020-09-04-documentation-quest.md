The TiKV Transactions SIG&lsquo;s documentation quest begins today! It is our first community activity. There are tons of bite-size tasks and some more involved work, with the goal of making TiKV&rsquo;s transaction components well-documented, easy to understand, and a go-to resource for learning how to implement distributed transactions.
Well-documented code is a joy to work with. Figuring out what undocumented code does is harder and takes longer, and you&rsquo;re more likely to miss some detail which causes a bug later on.
